support

support

Métodos que auxiliam as outras classes do framework.


String2Token

Converte uma string em tokens, esse método recebe uma string como parâmetro de entrada e retorna um lista de string: 

															
                                    print(pb.string2Token("essa frase sera separada em tokens"))
                                    #['essa','frase','será','separada','em','tokens']

token2String

Esse método retorna converte uma lista de tokens em uma string. Tem como parâmetro de entrada uma lista de string e retorna uma string: 

                            							
                                print(pb.token2String(['um','dia','serei','uma','string']))
                                #um dia serei uma string

splitInSentence

Este método separa as sentenças de um texto. Tem como parâmetro uma string e após processa-la retorna uma lista de strings. O algoritmo procura por sinais de pontuação, mas ele faz também o tratamento de abreviações, como por exemplo a palvra Def. que representa defesa. Existe uma lista em ./prebot/lang/portuguese/abbreviation.txt onde contém todas as abreviações segundo a ABNT, ABNT NBR 6032. 

                            							
                                print(pb.removeSpecialCharacter("chamando Dr. Hans Chucrute! chamando Dr. Hans Chucrute! Ola querida voê me chamou?"))
                                #['chamando dr. hans chucrute!', 'chamando dr. hans chucrute!', 'ola querida voê me chamou?']

Vetorização de Texto


Ngram

O algoritmo de ngram cria uma sequencia de n itens de uma string. Esse método tem 3 parâmetros, phrase que é uma string, que será processada, n que é o numero de itens e unity que é a unidade, por default o algoritmo trabalha com w que retorna palavras, mas você pode mudar informando c para retornar caractere. 

                            							
                                print(pb.ngram("me divido em 3 em 3",3))
                                #[['me', 'divido', 'em'], ['divido', 'em', '3'], ['em', '3', 'em'], ['3', 'em', '3']]
                                print(pb.ngram("me divido em 3 em 3 mas agora é com caractere",3))
                                #['me ', 'e d', ' di', 'div', 'ivi', 'vid', 'ido', 'do ', 'o e', ' em', 'em ', 'm 3', ' 3 ', '3 e', ' em', 'em ', 'm 3', '3 ', '3 m', ' ma', 'mas', 'as ', 's a', ' ag', 'ago', 'gor', 'ora', 'ra ', 'a é', ' é ', 'é c', ' co', 'com', 'om ', 'm c',' ca', 'car', 'ara', 'rac', 'act', 'cte', 'ter','ere']

Bag of Words

Neste modelo uma string é representada na contagem de suas palavras, retornando um dicionário: 

                            							
                                print(pb.bagOfWords("palavras e mais palabras, essa é minha composição"))
                                #{'palavras': 2, 'e': 1, 'mais': 1, 'essa': 1, 'é': 1, 'minha': 1, 'composição': 1}
