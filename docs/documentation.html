Instalação

Requisitos

O framework prebot requer alguns requisitos para poder funcionar corretamente. Mas não se preocupe é muito fácil consegui-los. Primeiramente você deve baixar o NLTK. O frmaework usa essa biblioteca para a realização de alguns métodos que envolvem processamento de linguagem natural.

Você pode baixa-los atraves do PIP, execute em seu terminal o seguinte comando: 

														
								pip install nltk

Uma vez instalado você precisará baixar o corpora Floresta e o algoritimo de steamming RSLP. Abra seu terminal e execute o comando python, para poder entar no interpretador, uma vez nele basta executar os seguinte passos: 

														
								import nltk
								nltk.download()

Pronto agora aparecerá uma janela de gerenciamento de download ou na própia linha de comando do terminal ira aparecer as opçoes. Basta agora baixar o corpora Floresta e o steamming RSLP.

Download Prebot

Acesse a página do projeto no github, clicando AQUI, e baxei o projeto da forma que achar mais conveniente. Ou pode executar o comando abaixo no seu terminal e clonar o repositório. 

														
								git clone https://github.com/mandala21/prebot.git

Pronto se tudo estiver ocorrido ok, você já pode usar o framework.
