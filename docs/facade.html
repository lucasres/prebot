Hello Word

Arquitetura

a classe prebotFacade, instância todas as outras classes do framework, por default. Mas você pode instanciar apenas alguns, para isso basta passar uma lista como parâmentro.

O código abaixo representanta um exemplo de instanciação: 

															
									from prebot.prebot import prebotFacade
                                    pb = prebotFacade()
                                    #todas as classes

A tabela abaixo mostra as opções que você pode usar na lista:

Parâmetro Classe que instânciada
normalizeText Classe responsável por métodos de normalização de texto.
stem Classe que contém métodos para stemming de palavras.
stopword Classe que trata da remoção de stop words.
tagger Classe responsável por fazer a POSTagger.
spellCheck Classe que contém métodos para realizar correção ortográfica.

Basta você passar uma lista com quais classes quer que sejam instanciadas na facade. Lembre-se que a classe support sempre é instanciada. 

							
								from prebot.prebot import prebotFacade
								
								pb = prebotFacade(['stem','tagger'])
								#além da support, contém a stem e tagger
